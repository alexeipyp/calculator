Тип приложения: Калькулятор

Требования к приложению:
	Одно поле для ввода и вывода данных
	Операции:
		Сложение
		Вычитание
		Умножение
		Деление
	Дополнительные операции: вычисление модуля числа
	Опции:
		Очистка поля ввода
	Добавить проверку правильности ввода числа пользователем

Минимальные требования:
	Два поля для ввода
	Одно поле для вывода: запрет ввода
